what is this?

ranjs is a library for generating various random variables in a robust and fast way.

how do I use it?

Browser

<script type="text/javascript" src="ranjs.min.js"></script>

Node

npm install ranjs

can I try it?

Sure, just take a look at the observable demo notebook here.

MCMC

ran.mc.MCMC(logDensity[, config[, initialState]])

Class implementing a Markov chain Monte Carlo sampler. MCMC samplers can be used to approximate integrals by efficiently sampling a density that cannot be normalized or sampled directly.
argtypedescription

configObjectObject describing some configurations. Supported properties:
  • dim: Dimension of the state space to sample. Default is 1.
  • maxHistory: Maximum length of history for aggregated computations. Default is 1000.
optional
initialStateObjectThe initial internal state of the sampler. Supported properties: x (the starting state), samplingRate (sampling rate) and internal for the child class' own internal parameters. optional

Carries our the initial warm-up phase of the sampler. During this phase, internal parameters may change and therefore sampling does not take place. Instead, all relevant variables are adjusted.

RWM

ran.mc.RWM()

Class implementing the (random walk) Metropolis algorithm. Proposals are updated according to the Metropolis-Within-Gibbs procedure as described in: Jeffrey S Rosenthal: Optimal Proposal Distributions and Adaptive MCMC, in Handbook of Markov Chain Monte Carlo.
