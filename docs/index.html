API

ran

Module for generating various random numbers.

ran.dist

A collection of generators for well-known distributions.

uniform

ran.dist.uniform(min, max[, n])

Generates some uniformly distributed random values.
argtypedescription
minnumberLower boundary.
maxnumberUpper boundary.
nnumberNumber of values to return. optional
returndescription
number ArraySingle value or array of random values.

argtypedescription
lambdanumberRate parameter.
nnumberNumber of values to return. optional
returndescription
number ArraySingle value or array of random values.
