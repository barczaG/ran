what is this?

ranjs is a library for generating various random variables in a robust and fast way.

how do I use it?

Browser

<script type="text/javascript" src="ranjs.min.js"></script>

Node

npm install ranjs

can I try it?

Sure, just take a look at the observable demo notebook here.

api

ran

Module for generating various random numbers.

ran.core

Core random generators and manipulators.

float

ran.core.float([min[, max[, n]]])

Generates some uniformly distributed random floats in (min, max). If min > max, a random float in (max, min) is generated. If no parameters are passed, generates a single random float between 0 and 1. If only min is specified, generates a single random float between 0 and min.
argtypedescription
minnumberLower boundary, or upper if max is not given. optional
maxnumberUpper boundary. optional
nnumberNumber of floats to generate. optional
returndescription
number ArraySingle float or array of random floats.

int

ran.core.int(min[, max[, n]])

Generates some uniformly distributed random integers in (min, max). If min > max, a random integer in (max, min) is generated. If only min is specified, generates a single random integer between 0 and min.
argtypedescription
minnumberLower boundary, or upper if max is not specified.
maxnumberUpper boundary. optional
nnumberNumber of integers to generate. optional
returndescription
number ArraySingle integer or array of random integers.

choice

ran.core.choice(values[, n])

Samples some elements with replacement from an array with uniform distribution.
argtypedescription
valuesArrayArray to sample from.
nnumberNumber of elements to sample. optional
returndescription
object ArraySingle element or array of sampled elements. If array is invalid, null pointer is returned.

shuffle

ran.core.shuffle(values)

Shuffles an array in-place using the Fisher--Yates algorithm.
argtypedescription
valuesArrayArray to shuffle. @return {Array} The shuffled array.

coin

ran.core.coin(head, tail[, p[, n]])

Flips a biased coin several times and returns the associated head/tail value or array of values.
argtypedescription
headobjectHead value.
tailobjectTail value.
pnumberBias (probability of head). If not specified, 0.5 is used. optional
nnumberNumber of coins to flip. optional
returndescription
object ArrayObject of head/tail value or an array of head/tail values.

ran.dist

A collection of random number generators for well-known distributions.

The general distribution generator, all generators are created using this class. The methods listed for this class are available for all distribution generators.

test

ran.dist.Distribution.test(values)

Tests if an array of values are from the specified distribution. For discrete distributions it uses the chi square test, whereas for continuous distributions the Kolmogorov-Smirnov test is used.
argtypedescription
valuesArrayArray of values to test.
